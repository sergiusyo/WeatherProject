Последовательность действий при создании скрипта виджета.

1) объявляем функцию создания новой карточки (getNewCard()); $card(это DOM-элемент) - див, и в него кладем всю карточку из html. создаваемая карточка будет объектом. Возвращаем из этой функции все элементы формы.
2) находим форму и создаем событие отправки и очищаем формы.
3) создаем карточку. newCard. Знак $ НЕ пишем, т.к. функция getNewCard возвращает ОБЪЕКТ, поэтому $ НЕ пишем. 
4) из инпута location берем значение $locationInput.value.trim(). И после отправки формы значение value в инпуте мы можем очищать!!
5) добавляем карточку, чтобы она появлялась вверху (prepend)
6 )добавим карточке анимацию. Меняем стиль формы и удаляем класс loading
7) создадим новую карточку. По умолчанию она пустая, нужна для того, чтобы помещать в нее новую карточку
8) при повторном добавлении карточки делаем проверку. Если карточка не пустая, то ей добавляется эффект стекла
9) находим и вставляем ключ API. Он необходим для предоставления доступа к серверам
10) создаем функцию для создания запросов на API. функция fetch асинхронная, т.е. ей нужно время для получения ответа от сервера.
11) обрабатываем запрос с помощью json-строки и возвращаем полученные данные. fetch(`https://api.openweathermap.org/data/2.5/weather?q=${location}&appid=${APIKEY}&units=metric`). units=metric - для того, чтобы получать значения теипературы в градусах по Цельсию. 
12) добавим в карточку необходимые параметры (температура, скорость ветра, влажность). Этот объект с данными получен с сервера по API
    далее мы получаем опиание погоды. weather это массив [0], обращаемся к его нулевому элементу
13) добавляем иконкую получаем данные по ней с сервера
14) меняем фон (бэкграунд) у иконки
